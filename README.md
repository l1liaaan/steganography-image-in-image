# Steganography LSB (Image in Image)

Notebook ini berisi implementasi steganografi menggunakan metode LSB untuk menyembunyikan gambar ke dalam gambar lain.

## Hasil
- Cover dan stego terlihat sama
- Secret berhasil diekstraksi kembali

## Cara Menjalankan
Buka file steganography.ipynb lalu Run All
